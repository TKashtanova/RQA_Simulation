
To run 'main.m' file, 

  1) Download the folder "PROGRAMS". 

  2) Download the "data" folder from
  https://drive.google.com/drive/folders/15F4miz6OdJwW2pXNJUvguteur_TguuLc?usp=share_link

  3) Place the "data" folder inside the "PROGRAMS" folder.

  4) Open the folder "PROGRAMS" in Matlab.

  5) Open 'main.m' in Matlab Editor and click "Run".
